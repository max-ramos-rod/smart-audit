# Template de ADR — movido

O template oficial e canônico de ADR do Smart Audit é **[`ADR_TEMPLATE.md`](ADR_TEMPLATE.md)**
(padronizado em Q7.4 / [ADR-0017](0017-modelo-unificado-de-evidencias.md)).

Use `ADR_TEMPLATE.md` ao criar uma nova ADR. Ele inclui o núcleo (Contexto · Decisão ·
Consequências · Alternativas descartadas), as seções de impacto "se aplicável" (schema · API ·
migração/rollout · operacional · critérios de aceite) e a **Checklist de Sincronização Documental
obrigatória**.

> Este arquivo é mantido apenas como redirecionamento para não quebrar links antigos.
